# Dominican Hidden Squeeze Research Project

## Tesis / Thesis

**ES:** Ni colapso. Ni comodidad. Una presión selectiva.  
**EN:** Not collapse. Not comfort. A selective squeeze.

La versión pública principal debe publicarse en español para audiencia dominicana.

- Sitio GitHub Pages: `docs/index.md`

## Boundaries

This project does not claim comfort.  
This project does not claim collapse.  
This project does not claim direct current household trade-down as proven.  
This project does not claim broad household cushioning.  
This project does not claim informal-worker cushioning as proven.

It supports a narrower interim conclusion:

**Ni colapso. Ni comodidad. Una presión selectiva.**
