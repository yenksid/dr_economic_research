# Investigación Dominican Hidden Squeeze

**Versión en inglés:** [README.en.md](README.en.md)

## Tesis central

**Ni colapso. Ni comodidad. Una presión selectiva.**

Este repositorio contiene la versión pública, bilingüe y basada en evidencia de una investigación sobre presión económica en República Dominicana.

La conclusión provisional no afirma que la economía dominicana esté colapsando. Tampoco afirma que los hogares estén cómodos. La lectura más fuerte, dentro de los límites de la evidencia disponible, es una presión selectiva: resistencia macroeconómica y amortiguación en el canal formal pueden coexistir con presión alimentaria, estrés de pequeños negocios, exposición informal no cerrada y falta de evidencia directa actual sobre sustitución de consumo.

## Límites activos

Este proyecto no afirma comodidad.  
Este proyecto no afirma colapso.  
Este proyecto no afirma sustitución directa y actual de consumo como hecho probado.  
Este proyecto no afirma amortiguación generalizada de todos los hogares.  
Este proyecto no afirma amortiguación probada para trabajadores informales.  

Afirma algo más estrecho:

**Ni colapso. Ni comodidad. Una presión selectiva.**
