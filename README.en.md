# Dominican Hidden Squeeze Research Project

**Spanish primary version:** [README.md](README.md)

## Core thesis

**Not collapse. Not comfort. A selective squeeze.**

This repository contains the public, bilingual, evidence-bounded version of a research project on economic pressure in the Dominican Republic.

The interim conclusion does not claim that the Dominican economy is collapsing. It also does not claim that households are comfortable. The strongest reading, within the limits of the available evidence, is a selective squeeze: macro resilience and formal-channel cushioning can coexist with food pressure, small-business stress, unresolved informal exposure, and missing direct current evidence on household trade-down.
