# Glosario

Este glosario explica siglas y términos técnicos usados en el proyecto.

## Siglas institucionales

| Término | Significado |
|---|---|
| BCRD | Banco Central de la República Dominicana. Fuente clave para datos macroeconómicos, laborales, monetarios y de precios. |
| ONE | Oficina Nacional de Estadística. Fuente clave para estadísticas oficiales, encuestas y datos socioeconómicos. |
| ENCFT | Encuesta Nacional Continua de Fuerza de Trabajo. Instrumento usado para medir empleo, informalidad, ingresos laborales y otros datos del mercado laboral. |
| TSS | Tesorería de la Seguridad Social. Relevante para trabajadores formales registrados y cotizantes. |
| DIGEPRES | Dirección General de Presupuesto. Fuente para ejecución presupuestaria y gasto público. |
| DGII | Dirección General de Impuestos Internos. Fuente para recaudación y actividad económica tributaria. |
| MEPyD | Ministerio de Economía, Planificación y Desarrollo. Fuente de contexto macro, social y de desarrollo. |
| MITUR | Ministerio de Turismo. Fuente para contexto turístico. |
| SB | Superintendencia de Bancos. Fuente para crédito, sistema financiero y estrés financiero. |

## Términos económicos

| Término | Explicación |
|---|---|
| Presión selectiva | Situación en la que la presión económica no afecta a todos por igual. Algunos grupos pueden estar protegidos mientras otros siguen expuestos. |
| Resistencia macroeconómica | Señales de fortaleza a nivel general, como actividad económica, entradas externas, empleo o turismo. No prueba comodidad en todos los hogares. |
| Canal formal | Segmento de trabajadores capturados por registros formales, como cotizantes de la seguridad social. No representa automáticamente a trabajadores informales. |
| Trabajador informal | Persona que trabaja fuera de los registros formales o sin protección/registro laboral completo. |
| Amortiguación | Capacidad de un ingreso, subsidio o apoyo para suavizar una presión de precios o gasto. |
| Compensación salario-alimentos | Comparación entre evolución salarial y presión de precios de alimentos para ver si un grupo pudo absorber parte del choque. |
| Sustitución de consumo | Cambio en la forma de comprar: marcas más baratas, menores cantidades, unidades más pequeñas, menos frecuencia o cambios de canasta. |
| Trade-down | Término inglés para sustitución hacia opciones más baratas o de menor tamaño/calidad/precio. En este proyecto no se afirma como probado para 2026. |
| Canasta | Conjunto de bienes usado para medir costo de vida, consumo o precios. |
| IPC | Índice de Precios al Consumidor. Mide variación de precios de una canasta de bienes y servicios. |
| MIPYME | Micro, pequeña y mediana empresa. Relevante para estrés financiero, costos de crédito y actividad comercial. |
| Remesas | Dinero enviado desde el exterior a hogares del país. Son importantes, pero el total macro no prueba automáticamente comodidad en todos los hogares. |
| Contexto fiscal | Datos sobre gasto público, transferencias, subsidios, deuda o presupuesto. |

## Términos metodológicos

| Término | Explicación |
|---|---|
| NOT_PROVEN | Sigue sin probarse. No significa falso; significa que la evidencia disponible no alcanza para afirmarlo. |
| REJECTED_BOUNDARY | Límite de inferencia rechazado. Significa que una conclusión no puede sacarse válidamente de esa evidencia. |
| Contexto solamente | Información útil para entender el entorno, pero insuficiente para probar una afirmación principal. |
| Puente de evidencia | Dato necesario para conectar una presión con una conclusión. Por ejemplo, para probar sustitución de consumo se necesita evidencia directa de comportamiento. |
| Alineación de período | La evidencia debe corresponder al mismo período que la afirmación. Un dato de 2025 no prueba automáticamente una afirmación de 2026. |
| Alineación de unidad | La unidad medida debe coincidir con la afirmación. Empleo no es lo mismo que ingreso; precio no es lo mismo que cantidad comprada. |
| Alineación de cobertura | La población cubierta por los datos debe coincidir con la población de la afirmación. Trabajadores formales no son todos los hogares. |
| Red-team | Ejercicio de atacar la tesis desde posiciones contrarias para ver si sobrevive sin exagerar. |
| Bibliografía pública clave | Lista curada de fuentes principales para lectura pública. |
| Registro local de auditoría | Registro más amplio y técnico que se conserva localmente, pero no necesariamente se publica completo por tamaño o ruido. |

## Línea final

**Ni colapso. Ni comodidad. Una presión selectiva.**
