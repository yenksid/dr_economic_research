# Evidencia faltante

Esta investigación llegó a una conclusión provisional publicable, pero no pretende cerrar toda la verdad económica.

La lista de evidencia faltante indica qué datos podrían cambiar, fortalecer o debilitar la conclusión.

## Frentes prioritarios

1. Tabla actual 2026 de ingresos/remuneraciones por formalidad o distribución de hogares.
2. Datos actuales de escáner minorista o canasta de supermercados.
3. Encuesta actual de gasto/volumen de consumo en hogares.
4. Encuesta actual con preguntas explícitas sobre sustitución.

## Regla

La evidencia futura solo puede mejorar una afirmación si pasa los requisitos de período, unidad, cobertura y método.
