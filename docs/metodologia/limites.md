# Metodología y límites

Este proyecto separa evidencia, contexto, inferencia y límites.

## Reglas

1. No convertir presión en comportamiento.
2. No generalizar datos formales a todos los hogares.
3. No convertir contexto laboral en prueba de ingresos.
4. No usar contexto de 2025 como prueba de 2026.
5. No usar precios por sí solos como prueba de sustitución.
6. No convertir ausencia en fuentes probadas en ausencia global.

## Resultado

La tesis no es una declaración partidaria ni una predicción cerrada. Es una conclusión provisional dentro de límites explícitos:

**Ni colapso. Ni comodidad. Una presión selectiva.**
