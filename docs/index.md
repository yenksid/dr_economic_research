# Ni colapso ni comodidad

## Evidencia de una presión selectiva sobre los hogares en República Dominicana

**Tesis central:** Ni colapso. Ni comodidad. Una presión selectiva.

[English version](./en/)

Este proyecto investiga si la economía dominicana puede describirse mejor como colapso, comodidad o presión selectiva. La conclusión provisional, dentro de los límites de la evidencia, es que la narrativa de colapso no se sostiene, pero la narrativa de comodidad tampoco.

La lectura más fuerte es una presión selectiva: resistencia macroeconómica y amortiguación en el canal formal pueden coexistir con presión alimentaria, estrés de pequeños negocios, exposición informal no cerrada y falta de evidencia directa actual sobre sustitución de consumo.

## Línea editorial

Este proyecto no afirma comodidad.  
Este proyecto no afirma colapso.  
Este proyecto no afirma sustitución directa de consumo como hecho probado.  
Este proyecto no afirma amortiguación generalizada de todos los hogares.  

Afirma algo más estrecho:

**Ni colapso. Ni comodidad. Una presión selectiva.**
